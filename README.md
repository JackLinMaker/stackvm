StackVM
=======
StackVM is a web-based virtual machine stack for managing, controlling, and
sharing VMS all through a browser.

Installation
============

Dependencies
------------

* [qemu](http://qemu.org)
* [npm](http://github.com/isaacs/npm)

Steps
-----

    git clone http://github.com/substack/stackvm.git
    cd stackvm
    npm install

Running
=======
    node server.js
